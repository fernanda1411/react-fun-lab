# To run project

### `yarn start`

# Working on a project

## create branch project

### `git checkout master`
### `git checkout -b  project-branch-name`


## Saving code to a project
### `git add .`
### `git commit -m 'message'`
### `git push origin project-branch-name`

## View project
### `git branch`

## Switching project
### `git checkout project-branch-name`
